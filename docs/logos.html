Usage Guidelines

Color Scheme: All logos use a consistent red gradient (#DC2626 to #991B1B) that aligns with OWASP-BLT brand identity.

Format: SVG format ensures scalability for any use case - from favicons to large banners.

Location: All logos are available in docs/assets/img/logos/

Attribution: Please maintain proper attribution to OWASP-BLT when using these logos.
